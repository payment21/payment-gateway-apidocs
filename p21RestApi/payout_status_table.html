Field Description
Payout Status The status of the requested payout.
Possible values:
Initialized: The Payout has been initialized
Awaiting_Verification: The payee needs to trigger the KYC verification process in Payment21
Verified: The payee has succesfully finished the KYC process in Payment21 and is ready to accept the payout
Awaiting_Funding: Awaiting the necessary fundings for processing the payout
Funded: The payout has been funded and is ready to be sent to the payee
Awaiting_Confirmation: The transaction was sent to the payee. Waiting for receipt confirmation
Finished: The Payout has been sent and confirmed by the payee
Expired: The Payout has expired
Canceled: The Payout has been canceled by the payee
Failed: The Payout has failed for other reasons
Code 1..7: Positive code from 1-7 means the Payout is still in progress and the final status is not reached yet
-2: When the payout status is Failed
-3: When the payout status is Expired
-4: When the payout status is Canceled
